\# Cách chạy Backend

mở terminal gõ:

python app.py

Lưu ý:

nếu ko chạy được thì hãy thử cài đặt flask-cors bang lệnh : pip install flask-cors

sau khi chạy xong thì hãy vào live server chạy file html để kiểm tra.
